# TT-Weather-Monitor
A real-time weather dashboard built specifically for Trinidad & Tobago, giving users a single-page snapshot of current conditions, hourly forecasts, a 5-day outlook, and automated weather alerts, all in one page.

## The Problem
Getting a quick yet comprehensive picture of T&T's current and upcoming weather means navigating multiple sources, such as the Met Office website, Windy, and various forecast apps that aren't by default calibrated for T&T and Caribbean geography. This dashboard pulls everything together in one place.

## Features

**Automated Weather Alerts** — monitors precipitation and wind data every 30 minutes and issues colour-coded warnings (Yellow, Orange, Red) and watches modelled on Trinidad & Tobago Meteorological Service (TTMS) criteria

**Live Satellite & Radar Maps** — four embedded Windy.com maps showing satellite view, rainfall, rainfall accumulation, and temperature

**Hourly Forecast** — next 8 hours of conditions including temperature, weather description, and rain probability

**5-Day Forecast** — daily high/low temperatures and weather conditions for the coming 5 days

## Important Disclaimer
The weather alerts generated by this app are **not official TTMS alerts**. They are automated assessments derived from Open-Meteo forecast data, evaluated against thresholds modelled on TTMS criteria. Always consult the [Trinidad & Tobago Meteorological Service website](https://www.metoffice.gov.tt/alert) for official weather warnings.

## Known Limitations
### TTMS Website Unreachable from Cloud Hosting
The original version of this app included two features powered by scraping the TTMS website: 

1. A prose forecast bulletin and
2. a text-to-speech audio readout of that bulletin.

Both features worked as intended when running locally. Upon deployment to Streamlit Cloud, the TTMS domain (metoffice.gov.tt) could not be resolved from the cloud hosting environment, likely due to the government server blocking or not supporting requests from cloud provider IP ranges. Both features were removed and replaced with Open-Meteo API data as a reliable alternative.

### Alerts Are Model-Based, Not Official
Alert thresholds are based on publicly available TTMS criteria applied to Open-Meteo forecast model output. They should be treated as a guide and a possible early indicator only.
